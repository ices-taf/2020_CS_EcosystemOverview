get_effort_map {icesVMS}R Documentation

Download fishing effort map data

Description

Download a data.frame of kw fishing hours by c-square and gear category averaged over 4 years.

Usage

get_effort_map(ecoregion, year = NULL)

Arguments

ecoregion

ICES ecoregion

year

which year to select (see details)

Details

The spatial data.frame contains average annual mw fishing hours, averaged over 4 years.

Value

a data.frame with a WKT column for the c-square polygons
