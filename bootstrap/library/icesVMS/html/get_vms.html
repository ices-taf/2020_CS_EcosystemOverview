get_vms {icesVMS}R Documentation

Download VMS data

Description

RESTRICED. Only core members of the ICES VMS datacall can acess this data. Download a data.frame of VMS data from the ICES VMS and logbook database.

Usage

get_vms(country, year, month, c_square, gear_code, metier, stat_rec,
  ices_area, ecoregion)

Arguments

country

country code

year

integer year

month

integer month

c_square

character 0.05 degree c-square name

gear_code

benthis gear code

metier

level 6 metier code

stat_rec

ICES statistical rectangle

ices_area

ICES area

ecoregion

ICES ecoregion

Value

a data.frame of VMS data
